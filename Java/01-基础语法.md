# Java 基础语法

## 一、Hello World

```java
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

> 文件名必须和 `public class` 后面的类名一致，即 `Hello.java`。

## 二、变量与数据类型

### 八种基本数据类型

| 类型      | 字节 | 范围              | 默认值      |
| ------- | -- | --------------- | -------- |
| byte    | 1  | -128 ~ 127      | 0        |
| short   | 2  | -32768 ~ 32767  | 0        |
| int     | 4  | 约 ±21 亿         | 0        |
| long    | 8  | 很大很大            | 0L       |
| float   | 4  | 单精度浮点           | 0.0f     |
| double  | 8  | 双精度浮点           | 0.0      |
| char    | 2  | 单个字符 'A'        | '\u0000' |
| boolean | 1  | true / false    | false    |

### 变量声明

```java
int age = 18;
double price = 9.99;
char grade = 'A';
boolean isStudent = true;
String name = "小林";  // String 是引用类型，不是基本类型
```

## 三、运算符

```java
// 算术运算
int a = 10 + 3;   // 13
int b = 10 / 3;   // 3（整数除法）
int c = 10 % 3;   // 1（取余）

// 比较运算（结果是 boolean）
boolean x = (5 > 3);   // true

// 逻辑运算
boolean y = true && false;  // 与
boolean z = true || false;  // 或
boolean w = !true;          // 非
```

## 四、控制流

### if-else

```java
int score = 85;
if (score >= 90) {
    System.out.println("优秀");
} else if (score >= 60) {
    System.out.println("及格");
} else {
    System.out.println("不及格");
}
```

### for 循环

```java
for (int i = 0; i < 5; i++) {
    System.out.println("第 " + i + " 次");
}
```

### while 循环

```java
int i = 0;
while (i < 5) {
    System.out.println(i);
    i++;
}
```

## 五、数组

```java
// 声明并初始化
int[] nums = {1, 2, 3, 4, 5};

// 访问元素
System.out.println(nums[0]);  // 1

// 遍历
for (int n : nums) {
    System.out.println(n);
}

// 长度
System.out.println(nums.length);  // 5
```

## 六、方法

```java
public class Calculator {
    public static int add(int a, int b) {
        return a + b;
    }

    public static void main(String[] args) {
        int result = add(3, 5);
        System.out.println(result);  // 8
    }
}
```
