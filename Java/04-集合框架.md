
# Java 集合框架

集合（Collection）是用来**存储多个对象的容器**，比数组更灵活——长度可变、提供丰富的操作方法。

## 一、集合体系结构

```
Iterable
└── Collection
    ├── List       → 有序、可重复
    │   ├── ArrayList    （数组实现，查询快）
    │   ├── LinkedList   （链表实现，增删快）
    │   └── Vector       （线程安全，已过时）
    ├── Set        → 无序、不可重复
    │   ├── HashSet      （哈希表实现）
    │   ├── LinkedHashSet（保持插入顺序）
    │   └── TreeSet      （自动排序）
    └── Queue      → 队列（先进先出）
        └── LinkedList、PriorityQueue 等

Map（独立体系，不属于 Collection）→ 键值对
├── HashMap         （最常用）
├── LinkedHashMap   （保持插入顺序）
├── TreeMap         （按 key 排序）
└── Hashtable       （线程安全，已过时）
```

## 二、List —— 有序、可重复

### ArrayList（最常用）

底层是**数组**，查询快，中间增删慢。

```java
import java.util.ArrayList;
import java.util.List;

List<String> list = new ArrayList<>();

// 添加
list.add("苹果");
list.add("香蕉");
list.add("橘子");
list.add("苹果");        // 可以重复

// 访问
System.out.println(list.get(0));     // 苹果
System.out.println(list.size());     // 4

// 修改
list.set(1, "葡萄");                  // 把索引 1 改成"葡萄"

// 删除
list.remove(0);                      // 按索引删
list.remove("橘子");                  // 按值删

// 判断
System.out.println(list.contains("葡萄"));   // true
System.out.println(list.isEmpty());          // false
```

### LinkedList

底层是**双向链表**，增删快，查询慢。

```java
LinkedList<String> linked = new LinkedList<>();
linked.add("A");
linked.addFirst("头部");      // 头部插入
linked.addLast("尾部");       // 尾部插入
System.out.println(linked);   // [头部, A, 尾部]
```

### ArrayList vs LinkedList

| 操作    | ArrayList | LinkedList |
| ----- | --------- | ---------- |
| 随机访问  | 快 O(1)    | 慢 O(n)     |
| 头部插入  | 慢 O(n)    | 快 O(1)     |
| 中间插入  | 慢 O(n)    | 慢 O(n)     |
| 内存占用  | 小         | 大（每个节点要存指针）|

> 💡 **大多数场景用 ArrayList 就行**，性能更优。

## 三、Set —— 无序、不可重复

### HashSet

底层是 **HashMap**，元素无序，去重靠 `hashCode()` 和 `equals()`。

```java
import java.util.HashSet;
import java.util.Set;

Set<String> set = new HashSet<>();
set.add("A");
set.add("B");
set.add("A");                 // 重复，不会添加
System.out.println(set);      // [A, B]（顺序不确定）
System.out.println(set.size());   // 2
```

### LinkedHashSet

保持**插入顺序**的 Set。

```java
Set<String> set = new LinkedHashSet<>();
set.add("C");
set.add("A");
set.add("B");
System.out.println(set);   // [C, A, B]（按插入顺序）
```

### TreeSet

**自动排序**的 Set，底层是红黑树。

```java
Set<Integer> set = new TreeSet<>();
set.add(3);
set.add(1);
set.add(2);
System.out.println(set);   // [1, 2, 3]（自动从小到大）
```

## 四、Map —— 键值对

### HashMap（最常用）

```java
import java.util.HashMap;
import java.util.Map;

Map<String, Integer> map = new HashMap<>();

// 添加 / 修改
map.put("苹果", 5);
map.put("香蕉", 3);
map.put("苹果", 10);     // key 重复，会覆盖

// 访问
System.out.println(map.get("苹果"));         // 10
System.out.println(map.getOrDefault("梨", 0)); // key 不存在时返回默认值 0

// 判断
System.out.println(map.containsKey("香蕉"));  // true
System.out.println(map.containsValue(10));   // true

// 删除
map.remove("香蕉");

// 大小
System.out.println(map.size());
```

### 遍历 Map 的三种方式

```java
Map<String, Integer> map = new HashMap<>();
map.put("a", 1);
map.put("b", 2);
map.put("c", 3);

// 方式 1：遍历 key
for (String key : map.keySet()) {
    System.out.println(key + " = " + map.get(key));
}

// 方式 2：遍历 entry（推荐，效率高）
for (Map.Entry<String, Integer> entry : map.entrySet()) {
    System.out.println(entry.getKey() + " = " + entry.getValue());
}

// 方式 3：forEach（Java 8+）
map.forEach((k, v) -> System.out.println(k + " = " + v));
```

### LinkedHashMap

保持**插入顺序**：

```java
Map<String, Integer> map = new LinkedHashMap<>();
map.put("c", 3);
map.put("a", 1);
map.put("b", 2);
// 遍历时按 c → a → b 的顺序
```

### TreeMap

按 **key 自动排序**：

```java
Map<String, Integer> map = new TreeMap<>();
map.put("c", 3);
map.put("a", 1);
map.put("b", 2);
// 遍历时按 a → b → c 的顺序
```

## 五、遍历集合

### for-each（最常用）

```java
List<String> list = List.of("A", "B", "C");
for (String s : list) {
    System.out.println(s);
}
```

### Iterator 迭代器

遍历时**安全删除元素**只能用迭代器：

```java
List<Integer> list = new ArrayList<>(List.of(1, 2, 3, 4, 5));
Iterator<Integer> it = list.iterator();
while (it.hasNext()) {
    int n = it.next();
    if (n % 2 == 0) {
        it.remove();   // 安全删除
    }
}
System.out.println(list);   // [1, 3, 5]
```

> ⚠️ 在 for-each 里直接用 `list.remove()` 会抛 `ConcurrentModificationException`。

### Stream（Java 8+）

```java
List<Integer> nums = List.of(1, 2, 3, 4, 5);

// 过滤 + 收集
List<Integer> evens = nums.stream()
    .filter(n -> n % 2 == 0)
    .collect(Collectors.toList());
System.out.println(evens);   // [2, 4]

// 求和
int sum = nums.stream().mapToInt(Integer::intValue).sum();

// 转大写
List<String> upper = List.of("a", "b", "c").stream()
    .map(String::toUpperCase)
    .collect(Collectors.toList());
```

## 六、常用工具类

### Collections

```java
import java.util.Collections;

List<Integer> list = new ArrayList<>(List.of(3, 1, 4, 1, 5));

Collections.sort(list);                  // 排序：[1, 1, 3, 4, 5]
Collections.reverse(list);               // 反转
Collections.shuffle(list);               // 随机打乱
int max = Collections.max(list);         // 最大值
int min = Collections.min(list);         // 最小值
```

### Arrays

```java
import java.util.Arrays;

int[] arr = {3, 1, 4, 1, 5};
Arrays.sort(arr);                        // 排序
System.out.println(Arrays.toString(arr));// [1, 1, 3, 4, 5]

// 数组转 List
List<Integer> list = Arrays.asList(1, 2, 3);
```

## 七、怎么选择合适的集合？

| 需求               | 推荐                                        |
| ---------------- | ----------------------------------------- |
| 有序、按索引访问         | **ArrayList**                             |
| 频繁头部/中间插入删除      | **LinkedList**                            |
| 去重、不关心顺序         | **HashSet**                               |
| 去重、保持插入顺序        | **LinkedHashSet**                         |
| 去重、自动排序          | **TreeSet**                               |
| 键值对、查询频繁         | **HashMap**                               |
| 键值对、保持插入顺序       | **LinkedHashMap**                         |
| 键值对、按 key 排序     | **TreeMap**                               |

## 八、注意事项

### 1. HashMap 不是线程安全的

多线程环境用 `ConcurrentHashMap`：
```java
Map<String, Integer> map = new ConcurrentHashMap<>();
```

### 2. 自定义对象作为 key/Set 元素时

**必须重写 `hashCode()` 和 `equals()`**，否则去重和查找会失效。

```java
public class Person {
    String name;
    int age;

    @Override
    public boolean equals(Object o) {
        if (this == o) return true;
        if (!(o instanceof Person)) return false;
        Person p = (Person) o;
        return age == p.age && Objects.equals(name, p.name);
    }

    @Override
    public int hashCode() {
        return Objects.hash(name, age);
    }
}
```

### 3. List.of() 返回的是不可变集合

```java
List<Integer> list = List.of(1, 2, 3);
list.add(4);   // ❌ 会抛 UnsupportedOperationException
```

要可变的话用：
```java
List<Integer> list = new ArrayList<>(List.of(1, 2, 3));
```
